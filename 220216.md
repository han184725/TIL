```python
def atoi(s):
    i=0
    for x in s:
        i=i*10+ord(x)-ord('0')
    return i
```

#고지식한 알고리즘(Brute Force)

```python
p="is"
t="This is a Book~!"
M=len(p)
N=len(t)
def BruteForce(p,t):
    i=0
    j=0
    while j<M and i<N:
        if t[i]!=p[j]:
            i=i-j
            j=-1
        i=i+1
        j=j+1
    if j==M:
        return i-M
    else:
        return -1
```
