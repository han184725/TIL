# Python chat-bot 

- 오픈소스: 제작자의 권리를 지키면서 누구나 코드 열람 가능
  - 공동 참여 프로젝트 등 생태계 구성
- 문법: 저장-조건-반복
  - 저장: box와 비슷(할당)
    - 숫자, 글자(따옴표), 참(True)/거짓(False) 가능
    - 변수, 리스트(0부터)([]), 딕셔너리({})
  - 조건: if : else:(Tab/Shift+Tab)
  - 반복: while, for
